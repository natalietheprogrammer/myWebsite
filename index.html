about me

Hi! I’m a front-end web developer based in the GTA of Ontario.

Through prior work as an engineer I came across several database applications and interesting programs, from which I developed a love of well-made, intuitive, and flexible software. This led me to my new career in programming.

I enjoy the challenge of creating responsive pixel perfect PSD to HTML conversions, but my true passion lies with the logic applied to JavaScript, the structure of APIs + Firebase, and the code reuse and code maintainability of React components.

Going forward I’m exited to apply my love of detail and logic towards building user friendly and eye-catching web applications.
